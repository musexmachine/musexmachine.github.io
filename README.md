## What this repo is

This repository powers the public Muse &amp; Machine site at `musexmachine.com`.

It is a plain GitHub Pages site with no build step:
- `index.html` for the landing page
- `404.html` and `500.html` for clean failure states
- `robots.txt`, `sitemap.xml`, `.nojekyll`, and `CNAME` for discovery and deployment

## Projects
